== Description ==

This plugin creates a function that can be placed anywhere on your wordpress site. This function makes an ad call to an aiMatch ad aserver. Included is also a widget that makes use of this function.

== Installation ==

Extract the zip file and just drop the folder aimatch-platform-connection in the wp-content/plugins/ directory of your WordPress installation and then activate the Plugin from Plugins page.

You can then go to the aiMatch Platform Connection usage page under plugins. This will explain how to use the plugin.
